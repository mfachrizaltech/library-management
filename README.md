# library-management
* library-management is the spring boot based rest api to managing the library 
* *Dependencies: spring-web spring-data-jpa
Use Lombok and Java Streams.

# Tech Steck
* Gradle 9.1
* Spring Boot 3.5.7
* Java 25

# Build The Project
* git clone https://github.com/mfachrizaltech/library-management
* cd library-management
* gradlew clean build
* gradlew bootRun

# Import The postman
* Goto postman directory
* Import library.postman_collection.json
* Run the request in library collection
